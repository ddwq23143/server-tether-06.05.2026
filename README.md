## 🧭 О проекте

Tether — это не очередной Telegram или WhatsApp. Это мой личный проект, в котором я учусь промышленной разработке на C# и .NET. Здесь нет задачи «собрать миллион пользователей». Задача — сделать надёжный, быстрый и честный сервер, который не врет про приватность.

Я устал от мессенджеров, которые собирают данные, тормозят и меняют правила на ходу. Поэтому я делаю свой.

## 🆙 Обновления

**11.05.2026** - Добавлено сохранение в БД. Для работы нужно создать таблицу `Messages` в Northwind.

```sql
CREATE TABLE Messages (
    Id INT IDENTITY(1,1) PRIMARY KEY,
    Username NVARCHAR(100) NOT NULL,
    Message NVARCHAR(MAX) NOT NULL,
    CreatedAt DATETIME NOT NULL
);
```

## 🔐 Принципы

- **Никакой слежки.** Я не собираю лишних данных. Только то, что нужно для работы: логин, время сообщения, текст.
- **Честность.** Код открыт. Любой может посмотреть, как он работает.
- **Простота.** Никаких навороченных анимаций и фоновых процессов. Только то, что нужно для общения.
- **Скорость.** Сервер написан на C#, который известен своей производительностью.

## 📌 Текущее состояние

Проект на начальном этапе. Сейчас сервер умеет:
- Принимать GET запросы с именем и сообщением которые сохраняются в .txt файле.

В планах:
- Сохранение в БД.
- JWT-аутентикация (логин и пароль).
- SignalR для обмена сообщениями в реальном времени.
- Деплой на VPS, чтобы Tether был доступен из любой точки мира.

## 📁 Структура (что будет)

- `Controllers/` — обработка запросов от клиентов
- `Data/` — работа с базой данных (SQLite)
- `Models/` — пользователи, сообщения, чаты
- `Services/` — JWT, хэширование паролей
- `Hubs/` — реальное время через SignalR
