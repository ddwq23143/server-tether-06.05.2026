## 🧭 О проекте

Tether - это мой личный проект, в котором я учусь промышленной разработке на C# и .NET. Здесь нет задачи «собрать миллион пользователей». Задача - сделать надёжный, быстрый и честный сервер, который не врет про приватность.

Я устал от мессенджеров, которые собирают данные, тормозят и меняют правила на ходу. Поэтому я делаю свой.

## 🔐 Принципы

- **Никакой слежки.** Я не собираю лишних данных. Только то, что нужно для работы: логин, время сообщения, текст.
- **Честность.** Код открыт. Любой может посмотреть, как он работает.
- **Простота.** Никаких навороченных анимаций и фоновых процессов. Только то, что нужно для общения.
- **Скорость.** Сервер написан на C#, который известен своей производительностью.

## 💻Как запустить

1. **Скопируй репозиторий:**
```bash
git clone https://github.com/ddwq23143/tether-messenger.git
cd tether-backend
```
2. **Настрой базу данных:**
   Проект использует SQL Server. Тебе нужно создать таблицы Users и Messages
3. **Настрой переменные окружения:**
   Создай файл `.env` в корневой папке проекта и скопируй туда настройки:
```sql
SERVER_IP=127.0.0.1 //Твой IPv4 или localhost
SERVER_PORT=5000
DB_SOURCE=.
DB_NAME= //твоя бд
DB_INTEGRATED_SECURITY=true
DB_TRUST_CERTIFICATE=true
```
4. **Установи библиотеки в корневой папке проекта:**
```bash
dotnet add package Microsoft.AspNetCore.OpenApi
dotnet add package Microsoft.Data.SqlClient
```
5. **Собери проект:**
```bash
   dotnet restore
   dotnet run
```
